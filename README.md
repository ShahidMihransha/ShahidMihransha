## ◈ Featured Projects

<details>
<summary><b>⚖️ TechVakkil — Privacy-First AI Legal Guidance Platform</b></summary>
<br>

> AI-powered legal guidance and lawyer marketplace for India — where the LLM never phones home.

TechVakkil democratizes access to Indian law by combining a locally-deployed LLM with a curated RAG pipeline over verified Indian legal statutes. Citizens describe their situation in plain language and receive structured legal guidance — with the right type of lawyer recommended when warranted. The entire AI stack runs on local hardware; no user data ever leaves the network.

<div align="center">

| Dimension | Details |
|-----------|---------|
| **Stack** | Flutter (web + mobile), FastAPI, Supabase (PostgreSQL), Ollama (Qwen3:8b), LlamaIndex, ChromaDB, HuggingFace BGE embeddings |
| **AI Architecture** | 5-type query classifier, 10-type document analyzer, multi-turn context gathering, RAG with `similarity_top_k=8` |
| **Scale** | 7+ Indian acts indexed, 13 verified lawyer profiles, persistent chat history, per-user document vault with Supabase Storage |
| **Security** | RLS on all tables, PII scrubbing on client + server, on-device ML Kit OCR, private storage bucket, zero external API calls |
| **Repository** | [`TechVakkil`](https://github.com/ShahidMihransha/TechVakkil) |

<details>
<summary><b>🚗 IoT-Enabled Smart Vehicle & Edge Battery Management System</b></summary>
<br>

> Gesture-controlled 4WD prototype with hardware-interrupt emergency braking and autonomous brownout prevention.

Built a full-stack IoT vehicle where Python + OpenCV + MediaPipe recognizes hand gestures on a host machine, transmits drive commands over UDP/TCP to an ESP32 Vehicle Control Unit, which executes motor control via L298N drivers with hardware-interrupt Automatic Emergency Braking. A companion Battery Management System uses ACS712 current sensors and voltage dividers for real-time power load calculation, with dynamic torque-limiting that autonomously prevents electrical brownouts before they happen.

<div align="center">

| Dimension | Details |
|-----------|---------|
| **Stack** | C++, Python, OpenCV, MediaPipe, ESP32, L298N motor driver, ACS712 current sensor, UDP/TCP networking |
| **Safety** | Hardware-interrupt AEB (not software-polled), dynamic torque curves based on live power budget, fail-safe motor cutoff |

</div>

</details>

<details>
<summary><b>⚡ Smart Energy Management System</b></summary>
<br>

> IoT smart circuit breaker with real-time energy monitoring and automated safety cutoffs.

Designed an ESP32-based energy monitoring system using ACS712 (current) and ZMPT101B (voltage) sensors that continuously calculates real-time power consumption. When unsafe electrical conditions are detected — overcurrent, overvoltage, or abnormal power factor — the system triggers automated relay-based safety cutoffs without human intervention.
